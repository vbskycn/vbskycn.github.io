挂载磁盘

说明

<code class="hljs python"><span class="hljs-comment">#/usr/bin/rclone mount DriveName:Folder LocalFolder </span>
下面整体代码中最上面的这条需要自己修改，不要直接复制，注意代码中有空格
DriveName       <span class="hljs-comment">#配置时填写的name</span>
Folder          <span class="hljs-comment">#网盘里要挂载的文件夹名</span>
LocalFolder     <span class="hljs-comment">#本地要挂载的文件夹绝对路径</span>
例：/usr/bin/rclone mount GD:VPS /home/GoogleDrive </code>

挂载

<code class="hljs perl"><span class="hljs-comment">#下面代码是一个整体，全部复制后粘贴运行</span>
/usr/bin/rclone mount DriveName:Folder LocalFolder 
 --<span class="hljs-keyword">umask</span> <span class="hljs-number">0000</span> 
 --default-permissions 
 --allow-non-empty 
 --allow-other 
 --transfers <span class="hljs-number">1</span> 
 --buffer-size <span class="hljs-number">64</span>M 
 --low-level-retries <span class="hljs-number">200</span>

<span class="hljs-comment">#如果你还涉及到读取使用，比如使用H5ai等在线播放，就在后面多加上以下三条参数</span>
--dir-cache-<span class="hljs-keyword">time</span> <span class="hljs-number">12</span>h
--vfs-<span class="hljs-keyword">read</span>-chunk-size <span class="hljs-number">32</span>M
--vfs-<span class="hljs-keyword">read</span>-chunk-size-limit <span class="hljs-number">1</span>G</code>

参数说明

<code class="hljs sql"><span class="hljs-comment">--transfers：该参数是最大同时传输任务数量，如果经常传输大文件，或CPU性能不佳，建议设置为单线程，也就是设置为“1”</span>
<span class="hljs-comment">--buffer-size：该参数为读取每个文件时的内存缓冲区大小，控制rclone上传和挂载的时候的内存占用</span>
<span class="hljs-comment">--low-level-retries：该参数为传输文件没速度的时候重试次数，没速度的时候，单个会自动睡眠10ms起，然后再重试</span></code>

设置自启动

下载并编辑脚本

<code class="hljs nginx"><span class="hljs-attribute">wget</span> https://www.moerats.com/usr/shell/rcloned && nano rcloned</code>

修改以下内容：

<code class="hljs python">NAME=<span class="hljs-string">""</span>  <span class="hljs-comment">#rclone name名，及配置时输入的name</span>
REMOTE=<span class="hljs-string">''</span>  <span class="hljs-comment">#远程文件夹，Google Drive网盘里的挂载的一个文件夹</span>
LOCAL=<span class="hljs-string">''</span>  <span class="hljs-comment">#挂载地址，VPS本地挂载目录</span></code>

设置自启

<code class="hljs sql">apt-get <span class="hljs-keyword">install</span> sudo -y
mv rcloned /etc/init.d/rcloned
chmod +x /etc/init.d/rcloned
<span class="hljs-keyword">update</span>-rc.d -f rcloned <span class="hljs-keyword">defaults</span>
bash /etc/init.d/rcloned <span class="hljs-keyword">start</span></code>
